Primitive Settings

      Primitive Type
      The shape of the volume to generate.

      Primitive Type Settings

      Radius / Width
      The radius or width of the selected object to be generated.

      VDB Settings

      Voxel Size
      The size of the voxels that make up the surface of the object. Lower values mean higher

      VDB Type
      A Signed Distance Field is a VDB representation of a surface, where voxels at the surface are set to a value of 0, and voxels surrounding that surface carry values that represent their distance from the surface. Voxels on the inside of the surface have a negative value while voxels on the outside have a positive value. A Fog Volume stores the same surface data as a Signed Distance Field except all the values of the voxels inside the surface are set to 1.

      Narrow Band Diameter
      Defines how many voxels wide the narrow band expands past the surface. The default values of 3 is usually more than enough.

      Fill Interior
      Instead of using a narrow band of values for the interior of the surface, this option fills all the voxels that make up the interior of the volume with values.

      Unsigned Distance Field
      The default SDF has positive values on the outside of the surface and negative values on the inside of the surface, this requires a surface to be air-tight. Use Unsigned Distance Field for surfaces that aren't air tight, thus making all the values positive.

      Grid Name
      The name of the grid that is being generated.

      Grid Type
      The data type and grid class of the grid being generated.

      Surface Count
      This number represents how many voxels are being drawn in the viewport to represent the volume.

      Voxel Count
      This number represents the total active voxels in the grid.

      Display Settings

      Display Shape
      The shape to draw each surface voxel as in the viewport. Square is the default and shows an accurate shaded version of the volume, it is also usually the fastest. Dot draws each voxel as a 1px dot with flat shading. Box is a 3D visualization of a voxel and it's size. Box (Lines) is the same a box but not filled in.
